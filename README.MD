# Proyecto NODE-JS reto técnico Sophos

Lamentablemente por las condiciones de la red y el equipo con el que tuve que trabajar, no alcancé a tener el proyecto funcionando con Mongo.

## Instalación 🔧

```
npm install
```

## Ejecutar 🚀
-- inicia la app --
```
npm start
```
-- crea registros ---
```
npm run test_create
```
-- prueba la creación --
```
npm run test_search
```
